Todo lo que hacemos es de código abierto

Nuestra plataforma consiste totalmente de componentes de código abierto y se basa en una plataforma sólida de programas libres (como GNU/Linux). Además, colaboramos de manera activa con el ecosistema para mejorarlo para todos.

Seguro y respetuoso de la privacidad

Como el código fuente está disponible para su revisión,
cualquiera (un investigador de seguridad, un usuario preocupado o un fabricante que incluye el sistema en sus equipos) puede verificar que el software es seguro y no filtra información personal.

Avisos de seguridad

Hecho para los desarrolladores

Tanto si su aplicación puede beneficiarse de una función o API nueva del sistema, como si tiene curiosidad por saber cómo se ha creado una prestación o una pauta de diseño existentes, tiene acceso completo a nuestro código fuente. Cópielo, aprenda de él, remézclelo, modifíquelo y redistribúyalo.

Participar

Respeto por la privacidad. En todos los ámbitos.

Sus datos le pertenecen exclusivamente a usted. No hacemos ofertas de publicidad ni recopilamos datos personales. Nos financiamos directamente gracias a nuestros usuarios, que pagan lo que desean por elementary OS y por las aplicaciones del Centro de aplicaciones. Así es como se debe hacer.

Nuestra normativa de privacidad

Sin escuchas

elementary OS le a ayuda a controlar el funcionamiento de las aplicaciones. Cuando una aplicación use el micrófono, se muestra un indicador para informar acerca de ello. Cuando una aplicación utilice demasiada energía, el indicador de energía lo marcará.

Permisos

Cuando una aplicación quiera acceder a los datos o dispositivos, deberá pedir permiso por adelantado. Revisamos todos los programas del Centro de aplicaciones para garantizar que utilicen adecuadamente los permisos, y siempre podrá revocarlos en la Configuración del sistema.

Mantenimiento

elementary OS mantiene ordenados los archivos temporales y los de la papelera de forma automática. Esto no solo ahorra espacio del dispositivo, sino que ayuda a asegurar que sus datos privados no le causen un trago amargo después.
